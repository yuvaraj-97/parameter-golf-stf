Going down

You accept STF as the root idea, then ask a narrower question: how should freezing be decided, what should a frozen token carry, and what global controls keep it stable?

Going sideways

You swap one answer for another at the same conceptual level. `stf-soft-freeze`, `stf-recurrence`, and `stf-quantization` are sideways alternatives, not parent-child steps.

Root

`baseline-repro` is the control branch. No STF. Every STF branch should be judged against it at the same budget slice.

Shared STF skeleton

All STF branches currently share: EMA of `||h_new - h_old||` as the freeze signal, layer warmup before freezing starts, and a minimum active-ratio guardrail to avoid collapse.
